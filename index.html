Hey, comece por aqui! Exemplos básicos para quem está iniciando seus estudos em estática.

Calcule o valor da força F

Para calcularmos o valor da força F, precisaremos das equações de equilíbrio: Σfx=0, Σfy=0 e ΣMp=0

Usando a promeira fórmula, teremos que Σfx=0 => Como não existe forças nesse eixo, devido ao apoio ser do primeiro gênero não utilizaremos essa primeira fórmula

Usando a segunda fórmula, temos que: Σfy=0 => F+20=P teremos então 2 icógnitas para somente 1 equação, seguiremos então para a terceira equação

Teremos então que:ΣM=0 => Adotaremos o ponto "P" para tirar o momento, também adotaremos o sentido horário como positivo, consequentemente, anti-horário como negativo =>
